* Introduction
:PROPERTIES:
:CUSTOM_ID: h-b0e00288-2568-43e4-8917-b211e91ccf82
:END:

Guile seems to be a Scheme which compiles to bytecode which is passed to
a JIT virtual machine. This pipeline allows for other "front-end
languages" like Emacs Lisp (or Pascal or Common Lisp or Haskell or...)
instead of Scheme.
